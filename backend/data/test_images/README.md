# Test Images

This directory contains test images for evaluating the Sentinel RAG system.

## Important Notes

- All images in this directory are from **public domain sources**
- Images are used for **demonstration and testing purposes only**
- Images are **NOT** real classified or sensitive military data

## Image Sources

### Radar Images
- NOAA Weather Radar (radar.weather.gov) - Public domain
- MarineTraffic AIS visualizations (marinetraffic.com) - For reference

### Satellite Images
- Sentinel-2 (ESA) - scihub.copernicus.eu - Open access
- Landsat (NASA/USGS) - earthexplorer.usgs.gov - Public domain
- FIRMS (NASA) - firms.modaps.eosdis.nasa.gov - Public domain

## File Naming Convention

```
[TYPE]-[NUMBER]-[DESCRIPTION].png

Examples:
- RADAR-001-weather-composite.png
- SAT-001-port-facility.png
```

## Adding New Images

1. Ensure the image is from a public domain or open-access source
2. Document the source in this README
3. Use the naming convention above
4. Add appropriate attribution

## Disclaimer

These images simulate tactical sensor data for demonstration purposes.
They do not contain actual classified or operational military information.
